# Gambling Demo dApp (Roulette + Prediction Market)

This demo dApp includes:

- a roulette-style number game
- a binary outcome prediction market

Main showcase contract:

- `contracts/MehCasinoAndPrediction.sol`

Reference import preserved:

- `contracts/DiceFate/`

## Intent

The game logic is intentionally simple and functional, with realistic beginner-level security weaknesses left in place for demonstration.

## Service Outputs

The three service deliverables are in:

- `services/01-smart-contract-audit.md`
- `services/02-audit-readiness-prep.md`
- `services/03-security-strategy-incident-response.md`
