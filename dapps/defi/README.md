# DeFi Demo dApp (Staking + Lending)

This demo dApp provides two simple modules:

- staking with time-based rewards
- collateralized borrowing and repayment

Main showcase contracts:

- `contracts/MockUSDC.sol`
- `contracts/MehStakingLending.sol`

Reference import preserved:

- `contracts/Never-Poor-Again/`

## Intent

The contracts are intentionally functional but security-light. They include common mistakes that are realistic in early-stage DeFi projects, making them good audit/readiness/strategy case studies.

## Service Outputs

The three service deliverables are in:

- `services/01-smart-contract-audit.md`
- `services/02-audit-readiness-prep.md`
- `services/03-security-strategy-incident-response.md`
